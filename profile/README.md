# Little Snitch — Portable Network Firewall & Connection Monitor

## About Little Snitch

Little Snitch provides a complete network monitoring and firewall environment for users who want to know exactly what leaves their machine. Little Snitch combines real‑time connection alerts, per‑application rules, and network traffic logging. Little Snitch supports silent mode, alert mode, and automatic rule learning. Little Snitch integrates with system network stack to intercept connections before they leave. Little Snitch enables users to block tracking domains, prevent malware callbacks, and audit application network behavior. Little Snitch runs on desktop platforms. Little Snitch is trusted by privacy advocates, security researchers, journalists, developers, and power users worldwide.
